# PhpMyAdmin Dark Theme

A dark material theme for PhpMyAdmin

## Setup

Go to folder phpmyadmin->themes and clone the repo
```bash
git clone https://github.com/naimjeem/PHPMyAdmin-DarkTheme.git
```
